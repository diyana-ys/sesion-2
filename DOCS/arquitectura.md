## Responsabilidades

- **Cliente**: Interfaz de usuario que consume la API.
- **Servidor API**: Gestiona las peticiones a TMDB y aplica lógica de negocio.
- **Base de datos**: Almacena cache y datos transformados.

> Nota: Este documento describe el sistema legacy. Actualizar según migraciones futuras.
